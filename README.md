# Scalable Todo App

The Scalable Todo App is a professionally architected full-stack solution designed for efficient task management. Leveraging the power of Next.js 16 for a responsive, high-performance frontend and a robust Express.js backend with TypeScript, it ensures both type safety and scalability. The system utilizes Prisma ORM for seamless database interactions and incorporates secure JWT authentication, making it a reliable foundation for modern web application development.

## Quick Start

1. **Start the Server** (Backend):
   ```bash
   cd server
   npm install
   npm run dev
   ```

2. **Start the Client** (Frontend):
   ```bash
   cd client
   npm install
   npm run dev
   ```
